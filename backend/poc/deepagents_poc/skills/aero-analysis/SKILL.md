# 气动分析技能

## 气动学基本结论
- 展弦比(AR)增大 → 诱导阻力降低 → 升阻比(L/D)增大
- 椭圆升力分布的 Oswald 效率因子 e ≈ 1.0(理论最优)
- 薄翼理论升力线斜率 ≈ 2π·AR/(AR+2) per rad
- 巡航升阻比典型值:滑翔机 > 30,运输机 ~15-20,战斗机 ~8-12

## 使用指引
当用户给出翼展/面积/迎角时:
1. 计算展弦比 AR = span²/S
2. 用薄翼理论估算 CL
3. 给出 L/D 的量级判断
4. 建议优化方向(通常:增大展弦比)
